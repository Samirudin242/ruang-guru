# Project Setup

## In Server Folder

<hr>

### Install All Dependencies

```
- npm install
```

### Database Setup

#### Create Database

```
- Sequelize db:create
```

#### Create Tabels And Columms

```
- Sequelize db:migrate
```

#### Input Datas Into Database

```
- Sequelize db:seed:all
```

### Run the Server

```
- npm start
```
